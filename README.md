# 🐦 Clone Twitter

Современное веб-приложение, вдохновлённое функциональностью социальной сети **Twitter (X)**.

Проект реализует backend социальной сети с REST API, позволяя пользователям публиковать твиты, подписываться друг на друга, ставить отметки «Нравится», взаимодействовать с контентом и управлять своим профилем. Приложение разработано с использованием **FastAPI** и построено с учётом принципов масштабируемости, мониторинга и удобства дальнейшего развития.

# 🚀 Начало работы

## Требования

Перед запуском проекта необходимо установить:

- Python 3.12
- Docker
- Docker Compose
- Poetry

---

## Сборка Docker-образа

```bash
docker build -t clone_twitter .
```

---

## Запуск приложения

Для разработки:

```bash
uvicorn backend.clone_twitter:app --reload
```

или, если используется Docker Compose:

```bash
docker compose up --build
```

После запуска API будет доступно по адресу:

```
http://localhost:8000
```
