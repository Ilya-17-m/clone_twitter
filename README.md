### Проект является клоном социальной сети Твиттер, как веб-приложение. В проете есть такие функции, как: подписки на других пользователей, добавление нового твита, добавление отметки нравится на твит и т.д. и т.п.

## Разработка:

### Требования:

#### Для установки и запуска проекта, необходим Python:3.12

### Установка зависимостей:
#### Для установки всез зависимостей и сбора Docker контейнера, выполните команду:
``docker build -t clone_twitter .``

### Запуск сервера:
#### Чтобы запустить сервер для разработки, выполните команду:
``uvicorn backend.clone_twitter --reload``

### Тестирование:
#### Наш проект был покрыт интерактивными тестами, для их запуска выполните комманду:
``pytest -v``
